## 解决步骤

### 1. 检查 API 密钥状态
1. 访问 [OpenWeatherMap API Keys](https://openweathermap.org/api_keys)
2. 登录您的账户
3. 检查密钥状态（Active/Inactive）

### 2. 生成新的 API 密钥
如果当前密钥无效，请：
1. 在 API Keys 页面点击 "Generate"
2. 给新密钥起一个名字（如 "Weather App"）
3. 复制新生成的密钥

### 3. 等待激活
新生成的 API 密钥可能需要：
- **几分钟到几小时** 才能完全激活
- 在此期间会显示 401 错误


### 5. 验证 API 密钥
